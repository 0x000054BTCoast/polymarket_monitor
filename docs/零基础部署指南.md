# polymarket-monitor 零基础部署指南（小白版）

> 目标读者：**0 编程经验**用户。  
> 目标：在你自己的电脑上把本项目跑起来，并能在浏览器看到页面。

---

## 你会得到什么

完成后你将可以打开：

- 前端页面：<http://localhost:3000>
- 后端接口：<http://localhost:8000/health>

如果你看到：

```json
{"status":"ok"}
```

就说明部署成功。

---

## 一、部署前准备（一次性）

请先安装这 3 个工具：

1. **Git**（用于下载项目）
2. **Python 3.11+**（用于后端）
3. **Node.js 18+（建议 20 LTS）**（用于前端）

### 1）检查是否安装成功

打开终端（Windows 用 PowerShell，Mac 用终端），分别输入：

```bash
git --version
python --version
node --version
npm --version
```

只要每条命令都能显示版本号就可以。

---

## 二、下载项目

在终端输入（把 `<你的仓库地址>` 替换成真实地址）：

```bash
git clone <你的仓库地址> polymarket-monitor
cd polymarket-monitor
```

---

## 三、配置环境变量（最简单方式）

项目根目录里已经有模板文件 `.env.example`。执行：

```bash
cp .env.example .env
```

> Windows PowerShell 可用：
>
> ```powershell
> Copy-Item .env.example .env
> ```

默认情况下**不需要改内容**，可以直接用。

---

## 四、启动后端（API）

在项目根目录执行：

```bash
make backend-install
make backend-run
```

看到类似 `Uvicorn running on http://127.0.0.1:8000` 就说明后端启动成功。

> 第一次安装依赖会慢一点，请耐心等待。

### 如果你的系统没有 `make`

也可以手动执行（Linux / macOS）：

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r backend/requirements.txt
cd backend
PYTHONPATH=. uvicorn app.main:app --reload --port 8000
```

Windows PowerShell 手动执行：

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r backend\requirements.txt
cd backend
$env:PYTHONPATH='.'
uvicorn app.main:app --reload --port 8000
```

---

## 五、启动前端（网页）

> 需要**新开一个终端窗口**（不要关掉后端窗口）。

进入项目根目录后执行：

```bash
make frontend-install
make frontend-run
```

看到类似 `Local: http://localhost:3000` 即表示前端成功启动。

---

## 六、访问页面并验收

1. 浏览器打开：<http://localhost:3000>
2. 再打开：<http://localhost:8000/health>

验收标准：

- 前端页面能打开
- `/health` 返回 `{"status":"ok"}`

---

## 七、常见问题（小白高频）

### 1）`make: command not found`

你没有安装 `make`，请按上面的“手动执行命令”步骤启动。

### 2）`python` 不是内部或外部命令

说明 Python 未安装或未加入 PATH。重新安装 Python，并勾选“Add Python to PATH”。

### 3）`npm` 不是内部或外部命令

说明 Node.js 未安装。安装 Node.js（建议 LTS 版本）后重试。

### 4）3000 或 8000 端口被占用

- 关闭占用端口的软件（可能是另一个前端/后端项目）
- 或临时改端口启动（进阶）

### 5）前端打开但没有数据

先检查后端是否还在运行，再访问：<http://localhost:8000/health>。

---

## 八、日常启动（第二次以后）

每次只需要：

- 终端 1（后端）
  ```bash
  cd polymarket-monitor
  make backend-run
  ```

- 终端 2（前端）
  ```bash
  cd polymarket-monitor
  make frontend-run
  ```

---

## 九、如何停止服务

在各自终端窗口按：

- `Ctrl + C`（Windows / Linux）
- `Control + C`（macOS）

即可停止。
